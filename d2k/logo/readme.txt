Max file created in 2014, saved in 2011. VRay 3.2 or later recommended, but earlier Vray versions can be used with minimal noticeable difference.
OBJ file Included, but shaders will have to be recreated manually.

For Post I used spi-anim from http://opencolorio.org/downloads.html. that's the only thing you need to get the same final result.

Planet textures were obtained from http://freebitmaps.blogspot.co.uk/, with usage rights purchased (by warman in IRC) to release us from the CC3.0 attribution requirements.
